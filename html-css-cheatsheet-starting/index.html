HTML Input Elements

HTML Tags

Tag Name Description
<textarea> Textarea Tag specifies an input field where the user can enter a long text.
<text> Text Tag specifies an input field where the user can enter a short text
<submit> Submit Tag specifies an input element that can be used to submit data to server

Tag Attributes

Attributes Name Description
Cols Columns Defines how many columns in the <textarea> input element
Rows Rows Defines how many rows in the <textarea> input element
name Name Specifies the name of an <text> input element
value Value Specifies the value of an <text> input element
